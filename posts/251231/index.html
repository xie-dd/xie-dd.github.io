ANSYS workbench + nCode 疲劳分析

资料：书籍《ANSYS nCode DesignLife 疲劳耐久性分析应用案例》P4

分析预览

对于下面的模型，在圆柱 1 面施加固定，t=1s 在圆柱 2 面施加集中力使圆轴弯曲，t=2s 在圆柱 2 面施加沿 X 的力矩使圆轴扭转。分析在两种载荷的交替作用下圆轴的寿命

步骤 1：获得静力分析结果

打开 Workbench，拖到一个静力分析模块，如下图 2 所示黄色部分，和一般的静力分析基本没有区别，下面介绍一些要点

(1) 材料方面，因为要进行疲劳分析，所以这里要选择一个带有疲劳数据的材料，我这里材料是 ncode 软件安装目录中nCode_matml.xml里面的Carbon Steel SAE1045_390_QT材料，如下图 3 所示

(2) 载荷是分两个时间步施加的，如下图 4 所示

步骤 2：进行疲劳分析

在完成结构分析之后，我们已经得到应力结果，可以用于 DesignLife 进行疲劳分析。在 Workbench 页面左侧的 toolbox 中有 8 个 DesignLife 分析系统。这里我们使用 nCode EN TimeSeries，如图 5 所示。

  • nCode EN/SN Constant 使用起始比例系数和终止比例系数乘以静力分析结果，生成循环应变/应力载荷
  • nCode EN/SN TimeSeries 使用静力分析结果和载荷历程文件匹配，创建疲劳应变/应力历程。
  • nCode SN VibrationPSD，该方法在频域范围内进行应力疲劳分析。
  • nCode SN VibrationSweptSine，处理扫频数据，适用于在不同频率扫动中的疲劳分析
  • nCode WeldShellSeam，专注于壳体焊缝的疲劳和强度分析，适用于薄壁结构& nCode
  • WeldSolidSeam，适用于更厚的固态焊缝分析，关注焊接接头的性能

双击 C5 打开，进入下图所示 DesignLift 系统

加载含应力的数据文件：点击 File—Open Data Files—选择项目文件夹，点击Scan Now —将 s3t 文件移到右边—点击Add To File List退出窗口

加载载荷： 将 DesugnLift 界面中 Available Data 中的shaft_loads.s3t文件拖到 TimeSeries_Input 窗口中。点击 Simulation_Input 窗口和 TimeSeries_Input 窗口的 Display 可以看到模型与载荷历程。

将仿真结果映射至时间序列：右键单击 StrainLife_Analysis 模块，选择 Edit Load Mapping，弹出窗口如下图所示。这里我们不用更改，系统自动配置是正确的。其中 Time 1 对应弯曲载荷的结果，Time 2 对应扭矩载荷的结果，点击 OK 退出该窗口。

求解： 在 DesugnLift 界面，点击最上面的 Run 按钮，执行求解。

后处理： 当求解完成以后，Fatigue_Results_Display 显示损伤云图。Data Values Display 以表格形式显示分析结果，点击列头可以排序，可以看到最小的寿命为1.949E+4
